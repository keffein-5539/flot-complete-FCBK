# 🚀 relaydev

> API gateway for microservices

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g relaydev
relaydev init my-project
```
